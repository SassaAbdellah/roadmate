Overview of Testing :

  1. Unit test: Programmers test their own classes during the coding.
  2. Integration: After one prototype cycle finish, one person is doing integration test:
    • interface
    • contents
    • security
    • navigation
    • validation
  3. System test: Test RoadMate Windows , Mac, Linux
  4. Usability test: choose different group of users, observe users performance, record users feedback.
  5. Stress test: use tools make large amount requests and record performance data.

 

Test tools:
use Selenium IDE ,which is implemented as a FireFox extension, and allows us to record, edit, and debug tests.
